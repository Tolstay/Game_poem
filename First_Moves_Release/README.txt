==============================
    First Moves 游戏分发指南
==============================

📦 发送给朋友时需要的文件：
✅ First_Moves.exe        (主程序文件)
✅ First_Moves.pck        (游戏资源包)

🚨 重要事项：
1. 这两个文件必须放在同一个文件夹内
2. 不能重命名这些文件
3. 双击 First_Moves.exe 启动游戏

🎮 游戏操作：
- 鼠标移动：角色跟随
- 长按鼠标左键：摘除花瓣/果实
- 空格键：生成新的枝条
- 目标：摘除所有花瓣

⚠️ 如果游戏无法启动：
1. 确保两个文件在同一文件夹
2. 右键以管理员身份运行
3. 检查防病毒软件是否拦截

📧 如有问题，请联系开发者
